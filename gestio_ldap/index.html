    Gestio del LDAP

    En aquest apartat, s'explica breument com gestionar el servidor LDAP utilitzant les comandes bàsiques: add, delete i modify. Aquestes comandes permeten afegir, eliminar i modificar entrades a la base de dades LDAP.

    ldapadd

    Per afegir una nova entrada a la base de dades LDAP, utilitzeu la comanda ldapadd. Aquesta comanda requereix un fitxer LDIF (LDAP Data Interchange Format) que defineix l'entrada que voleu afegir.

    Fitxer user.ldif

    dn: uid=cracked,ou=usuaris,dc=xanax,dc=cat
objectClass: inetOrgPerson
objectClass: posixAccount
objectClass: top
uid: Cracked
cn: Cracked
sn: 123
givenName: Cracked123
uidNumber: 1004
gidNumber: 1009
homeDirectory: /home/cracked
loginShell: /bin/bash
userPassword: {SSHA}123
mail: cracked@xanax.cat

    Per esborrar una entrada d'un domini LDAP mitjançant un fitxer .ldif, podeu utilitzar la comanda ldapmodify. Aquest mètode és útil quan voleu gestionar múltiples esborrats o necessiteu un major control sobre les operacions. A continuació, s'explica com fer-ho:

    El fitxer .ldif ha de contenir les instruccions per esborrar l'entrada. Per exemple, per esborrar l'usuari alu1 del domini xanax.cat, creeu un fitxer anomenat esborrar_alu1.ldif amb el següent contingut:

    esborrar_alu1.ldif:

    dn: uid=alu1,ou=usuaris,dc=xanax,dc=cat
changetype: delete

    Un cop creat el fitxer .ldif, executeu la següent comanda per aplicar l'esborra

    ldapmodify -x -D "cn=admin,dc=xanax,dc=cat" -W -f esborrar_alu1.ldif

    ldapmodify

    ldapmodify és una eina que permet modificar entrades en un servidor LDAP mitjançant fitxers .ldif. Aquests fitxers contenen instruccions per afegir, modificar o esborrar atributs i entrades.

    Modificar un usuari

    En aquest exemple veurem com modificar algun camp de un usuari

    Fitxer ldif

    dn: uid=alu1,ou=usuaris,dc=xanax,dc=cat
changetype: modify
replace: telephoneNumber
telephoneNumber: 123456789
-
replace: mail
mail: alu1@xanax.cat

    Executa esta comanda

    ldapmodify -x -D "cn=admin,dc=xanax,dc=cat" -W -f modificar_alu1.ldif

    Verifició

    ldapsearch -xLLL -b "uid=alu1,ou=usuaris,dc=xanax,dc=cat" telephoneNumber mail

    Modificar varios parametres

    En aques exemple els objectius son:

    • Modificar el número de telèfon.

    • Afegir un correu electrònic i la ciutat de residència.

    • Moure l'usuari a la UO asix.

    Fitxer ldif

    dn: uid=alu1,ou=usuaris,dc=xanax,dc=cat
changetype: modify
replace: telephoneNumber
telephoneNumber: 987654321
-
add: mail
mail: alu1@xanax.cat
-
add: l
l: Barcelona
-
changetype: modrdn
newrdn: uid=alu1
deleteoldrdn: 1
newsuperior: ou=asix,dc=xanax,dc=cat

    Executa esta comanda

    ldapmodify -x -D "cn=admin,dc=xanax,dc=cat" -W -f modificar_alu1_complex.ldif

    Verifició

    ldapsearch -xLLL -b "uid=alu1,ou=asix,dc=xanax,dc=cat"
